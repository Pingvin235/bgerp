= Plugin Log
:toc:

[[about]]
== About
The plugin collects *Action Log*  with all performed executed actions and provides viewing the data with filtering.

NOTE: You can see the plugin enabled in <<../../../kernel/install.adoc#demo, Demo System>>.

[[setup]]
== Setup
Enable the plugin in <<../../../kernel/setup.adoc#config-plugin, server config>>.
[source]
----
log:enable=1
----

[[usage]]
== Usage
Log view tool is available in menu *Administration / Action Log*.
