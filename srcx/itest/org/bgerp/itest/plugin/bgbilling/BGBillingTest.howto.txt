# В карточке процесса
- Привязан договор test с ID=1
- При клике мышью по номеру договора открывается его карточка

# После удаления привязанного договора
- Он предлагается при привязке из буфера обмена, либо может быть привязан по номеру

# После повторной привязки договора и клике мышью по заголовку процесса
- Должны скопироваться параметры, указанные в JEXL скрипте конфигурации типа процесса
